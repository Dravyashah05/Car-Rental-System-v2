### Step 1: Explore Home

- Click "Browse Cabs" or use search
- Apply filters (price, rating, seats)

## 🚀 Deployment Steps

### Option 1: Vercel (Easiest)
1. `npm run build`
2. Create account at vercel.com
3. Deploy `dist/` folder
4. Done! 🎉

### Option 2: Netlify
1. `npm run build`
2. Create account at netlify.com
3. Drag & drop `dist/` folder
4. Done! 🎉

### Option 3: Any Static Host
1. `npm run build`
2. Upload `dist/` folder
3. Configure to serve `index.html` for all routes
4. Done! 🎉
